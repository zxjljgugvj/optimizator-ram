# Optimizator-Windows
You can optimize your CPU and clean your RAM. 
When you open "Optimizator-RAM.bat" you need to disable UAC, that's why RAMMAP will always ask to run as an Administrator.
It only works through the command line always run this .bat as an Administator.

"Optimizator-RAM.bat":
1. Disables Windows Update.
2. Clears the TEMP folder.
3. Clears the local DNS resolver cache.
4. Clears RAM, also using RAMMap.
5. Optimize CPU.
6. Disable not important services.
7. Enable High-Performance power mode.

"Set-To-Default.bat":
1. Re-enables Windows Update, Scheluded tasks, Services.
